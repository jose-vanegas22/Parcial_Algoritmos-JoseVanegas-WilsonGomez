# Parcial Algoritmos - Jose Vanegas - Wilson Gomez

## Descripción
Solución de 2 ejercicios de HackerRank para el examen práctico de Algoritmos.
Los ejercicios fueron tomados y ejecutados directamente en la plataforma HackerRank.

## Ejercicios

### 1. CamelCase (String / Greedy)
Dado un string en camelCase, contar cuántas palabras contiene.

**Complejidad:** `O(n)` — se recorre el string una sola vez

### 2. Maximum Perimeter Triangle (Sorting / Search)
Dado un arreglo de palos, encontrar el triángulo de mayor perímetro posible.

**Complejidad:** `O(n log n)` — dominada por el ordenamiento `.sort()`

## Cómo ejecutar
Los archivos fueron desarrollados y probados en **HackerRank**.

1. Ingresar a [HackerRank](https://www.hackerrank.com)
2. Buscar el ejercicio correspondiente
3. Copiar el código del archivo `.py` en el editor
4. Ejecutar con el botón **Run Code**

## Requisitos
- Cuenta en HackerRank
- Python 3
